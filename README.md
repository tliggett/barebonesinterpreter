# barebonesinterpreter
Interprets Bare Bones programming language

## Installation:
You must have make and gcc installed on your machine.

### Install make/gcc
#### OSX (Mac)
brew install make
brew install gcc
#### Linux
##### ubuntu/debian
sudo apt install build-essential
##### fedora
yum groupinstall 'Development Tools'

### Install barebones
make barebones
sudo make install
